== Description ==

This plugin allows you to insert a Merch38 widget into your website.

This widget renders an e-commerce widget that you can use to create a dynamic t-shirt product in a few minutes and easily sell the merchandise online.

== What is Merch38? ==

[Merch38](https://merch38.com/merchandise-for-web/) is a B2B platform for printing merchandise.

You can start selling personalized merch with zero investment and no inventory.

With easy-to-install web plugins and complete customer fulfillment, Merch38 allows customers to make t-shirts with designs that truly excite them, instead of compromising on someone elseâ€™s design.

== How Merch38 for WP works? ==

This plugins essentially is an easy way to embed your [Merch38 campaign](https://app.merch38.com/campaigns) into your WordPress website.

It uses Merch38 API Key to fetch your list of campaigns and offers a shortcode for each campaign.

You can change the value of a shortcode attribute and easily offer your personalized merchandise for your users.


== Installation ==

1. Upload `merch38-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Merch38 plugin menu
1. Follow the instructions to setup the API key for connecting with Merch38 account.
